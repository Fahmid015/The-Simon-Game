Press any Key to Start

How To Play:

After pressing a key to start the game, the computer will flash a certain color which the user will have to click. As the levels progress, the user will have to remember and click the previous clicked colors along with the new color showed by the computer as a pattern to progress further levels. The computer will only show the last color needed to be clicked in the pattern. Upon clicking the wrong color/ pattern, the game will end.
